# 字符串操作（一）
1、去除字符串开头的空白

```
function removeBeginBlank(str) {
    return str.replace(/^\s*/g, '');
}

var str = '     test string';
removeBeginBlank(str);
```

2、去除字符串结尾处的空白

```
function removeTrailBlank(str) {
    return str.replace(/\s*$/g, ''); 
}
var str = "let go     ";
removeTrailBlank(str);
```
